THE HTML ATTRIBUTES



Attributes: an attribute is used to add more information to an element because an
attribute must be written inside the opening tag of an element. An attribute is
composed by 2 parts:- The name of the attribute- The value of the attribute inside quotations marks
Example: href = “index.html” -> the name is “href” and the value is “index.html”
